    JavaScript를 활용한 동적 구현 웹 디자인 2023.07.26 - 08.02

    OVERVIEW

    독일의 미술과 공예, 사진, 건축 등을 교육한 학교로
    바우하우스의 선언문과 작품을 기반으로 제작한 웹 페이지이며 그 철학을 바탕으로 디자인
    Javascript를 활용해 동적인 웹으로 제작

    사용 기술

    HTML5 / CSS3 / Javascript / Figma

    FONT

    COLOR

    #단순한 #원색의 #감각적인

    ELEMENTS

    Grayscale과 Blending mode

    바우하우스의 인물과 건물, 제품등의 실제 요소를
    흑백으로 나타내어 원색과의 색 대비를 극대화 시키고
    mix-blend-mode를 사용하여 컬러와 어우러지도록 표현.
    오늘날까지도 다양한 분야에 많은 영향과 영감을 주는 바우하우스의
    철학과 역사를 느낄수 있도록 디자인.

    기본도형의 사용

    바우하우스의 교육 철학은 효과적이고 효율적인 디자인을 중심으로
    했기 때문에 감각적이고 단순한 형태와 색감을 사용.
    간결한 모양과 제한된 색을 이용해 색다른 디자인을 만들어내는
    가능성을 보여주는 것이야말로 바우하우스다운 디자인이라 생각한다.

    문제 & 해결

    마우스를 따라 움직이는 눈동자 상단과 하단에 바우하우스의 대표적인 ‘눈’포스터를 배치. 눈동자 요소 안의 큰 원과 작은 원에
    각각 다른 ‘mousemove’ 이벤트를 적용시켜 마우스를 자연스럽게 보는 눈동자를 표현하고 싶었으나,
    창 크기 변화에 따른 위치 조절 및 움직임의 제어가 뜻대로 되지 않아 적용시키지 못했다.

    스크롤 트리거. Pin 고정 사용
     이전 사용해 보았던 scrollTrigger를 이용하여 대부분의 스크롤 이벤트는 적용 시킬 수 있었고
    pin 속성을 새롭게 적용. 애니메이션이 끝날 때까지 일부 영역을 고정시켜 보았다.

    교차되는 스크롤 scrollTop()으로 수직 스크롤 값을 구하고 여기에 *-1 을 적용시켜
    역으로 교차되는 스크롤 이벤트를 만들어 보려고 했으나 적용이 되지 않았다.
    결국 scrollTrigger를 이용해 자연스럽지는 못하지만 어느 정도의 교차되는 스크롤을 표현 할 수 있었다.
